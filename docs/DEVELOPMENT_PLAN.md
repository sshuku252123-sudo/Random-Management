## 重要なノート

### 技術スタック
- **フロントエンド**: Next.js 14, React 18, Tailwind CSS, TypeScript
- **バックエンド**: Node.js, Express, TypeScript
- **データベース**: Supabase (PostgreSQL)
- **認証**: Supabase Auth（実装予定）

### 環境設定
- **開発環境**: http://localhost:3000 (Frontend), http://localhost:3001 (Backend)
- **本番環境**: 未設定（Vercel/その他の選定必要）
